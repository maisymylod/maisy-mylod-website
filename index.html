monte carlo

100,000 paths one option price.

Geometric Brownian motion for a European call. Each pale line is a simulated path; the pink one is the latest. Average their payoffs, discount to today, and you get a Black-Scholes-matching price.

quantlib_mm/pricer.py 
from quantlib_mm import simulate, european_call

paths = simulate(s0=100, mu=0.08, sigma=0.22,
                 n_paths=100_000, n_steps=252)
price, std = european_call(paths, k=100, r=0.04)
# price ≈ 10.45, std ≈ 0.04

spectral graph

n nodes k communities, no labels.

Build the graph laplacian L = D − A, take its bottom eigenvectors, k-means the rows. Communities pop out without ever telling the algorithm what they are.

spectral_cluster.py 
import numpy as np
from scipy.linalg import eigh

L = D - A                              # graph laplacian
vals, vecs = eigh(L)                   # spectrum
labels = kmeans(vecs[:, 1:k+1], k)   # cluster rows

persistent homology

point cloud Betti(0,1,2), as scale grows.

Grow a ball around each point. Watch the Vietoris-Rips complex form, holes appear and die. The features that persist across scale are the real shape of the data.

rips_complex.py

result = ripser(points, maxdim=2)
